If you encounter the `"Please obtain the RESTful API ID and key first"` error message when configuring aPaaS in the Console, please take the following steps:

1. Log in to Console, click the account name in the upper right corner, and select **RESTful API** in the the drop-down menu.

1. Click **Add Key** to get Customer ID and Customer Secret. After obtaining it, continue to configure aPaaS.
