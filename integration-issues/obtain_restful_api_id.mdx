When configuring aPaaS on the Shengwang console, if you encounter an error message "Please obtain the RESTful API ID and key first", please do the following:

1. Log in to the Shengwang console , click the account name in the upper right corner, and enter the drop-down menu RESTful API page.

1. Click Add Key to get Customer ID and Customer Secret. After obtaining it, continue to configure aPaaS.
