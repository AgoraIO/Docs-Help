The Agora Video SDK is compatible with VR (Virtual Reality) headsets on the following operating systems:

- Android
- iOS
- Windows (non-Universal Windows Platform versions only)

Supported VR headsets include the following:

- Google Cardboard
- Oculus Rift
- Oculus Go
- Oculus Quest
- Vive Pro
- Vive Focus Pro
- Vive Focus 3
